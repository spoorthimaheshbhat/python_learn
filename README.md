## Rules For This Journey

You must write code yourself

* _No copy-paste learning._

You must explain your own code

* _If you can explain it, you understand it._

You must refactor old code

* _Professional growth happens here._

You must push regularly to GitHub

* _Consistency matters more than intensity._


We prioritize:

- clarity
- maintainability
- Pythonic thinking
- engineering habits

* _not “just making code work.”_
